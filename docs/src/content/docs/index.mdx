## Features

<CardGrid stagger>
  <Card title="Native SDK" icon="rocket">
    Uses the official Telegram Login SDK for iOS and Android with native OAuth
    flow.
  </Card>
  <Card title="Universal Links" icon="link">
    Supports iOS Universal Links (iOS 17.4+) and App Links on Android.
  </Card>
  <Card title="Easy Integration" icon="setting">
    Automatic URL callback forwarding — no AppDelegate or Activity wiring
    required.
  </Card>
  <Card title="Cancellation" icon="close">
    Cancel in-flight login attempts with `cancelLogin()`.
  </Card>
</CardGrid>
